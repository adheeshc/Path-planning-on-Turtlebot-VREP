=================================
FILE DESCRIPTIONS
=================================
1) final.m - used to perform A star method
2) c2g.m - calculates cost to go
3) checkposition.m - checks obstacle collisions
4) diffconstraints.m - defines the differential constraints
5) get_velocities.m - gets resolution from angular velocities
6) new_node.m - creates nodes according to A star algorithm
7) remApi.m - used for creating API for VREP
8) remoteApiProto.m - used for creating API for VREP
9) remoteApi.dll - used for creating API for VREP
10) rrl_map.ttt - VREP Environment Map with Turtlebot
11) simpleTest.m - used for creating API for VREP
12) vrep_run.m - Code to be run for implementing A star in VREP

=================================
RUN INSTRUCTION
=================================
1) Make sure all files are in the same folder
2) Run rrl_map.ttt using VREP. Start the simulation
3) Run vrep_run.m using MATLAB, run the code
4) Follow instructions
5) Wait for Matlab to finish calculation
6) Switch to VREP to watch TurtleBot moving
